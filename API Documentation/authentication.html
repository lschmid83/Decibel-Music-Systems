Authentication

Decibel’s API uses App_ID and App_Key_Pair authentication for simplified access and authorization. Using App_ID mode is simple. Check your account details to find your access string for your app and begin!

Do I need to authenticate?

Yes, Decibel’s API is only accessible through the use of app_id and require the use of an app_key. Each application you create should use a new app_id and app_key. These ids and keys are unique to the application and should be stored securely.

How do I find my App_ID and App_Key?

In order to receive an App_ID, you must do the following:
  1. For new users: Sign up for a Decibel user account
    • Upon registration, you will be asked to name and describe your application. This will generate your first app_id
    • To retrieve the app_key, confirm your account registration and follow the App Key retrieval instructions: Dashboard Applications select App Name View
  2. For existing users: login to Decibel account
    • Follow the same App Key retrieval instructions for an existing app: Dashboard Applications select App Name View
  3. To create a new app, My Applications Create New Application View

How do I call the API?

API Address
Decibel API
http://rest.decibel.net/v2

The Rest API is stateless, which means you need to identify yourself with each call.

On each call to the API you will need an Application ID, Application Key and Timestamp included in your HTTP request header. An Application ID and Key are provided when you register

  • DecibelAppID = {Your Application ID}
  • DecibelAppKey = {Your Application Key}
  • DecibelTimeStamp = 20120402 13:25:07(yyyyMMdd HH:mm:ss) - a time stamp

The HTTP Request Header should look like this:

GET /v2/albums?artistName=fleetwood%20mac HTTP/1.1
Host: api.decibel.net
Connection: keep-alive
User-Agent: Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/535.11 (KHTML, like Gecko) Chrome/17.0.963.83 Safari/535.11
Accept: text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8
Accept-Encoding: gzip,deflate,sdch
Accept-Language: en-GB,en-US;q=0.8,en;q=0.6
Accept-Charset: ISO-8859-1,utf-8;q=0.7,*;q=0.3
DecibelAppID: {Your Application ID}
DecibelAppKey: {Your Application Key}
DecibelTimestamp: 20120402 13:25:07
