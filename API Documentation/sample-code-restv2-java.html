Calling the API
Back to top

The WebClient package included in the sample project provides methods for receiving data from a resource identified by a URI.

The Rest API is stateless, which means you need to identify yourself with each call.

The WebClient instance does not send optional HTTP headers by default. On each call to the API you will need an Application ID, Application Key and Timestamp included in your HTTP request header. To make this easier a function called setDecibelRequestHeaders(WebClient, AppID, AppKey) is defined in the Util.java class.

The following code example sets the WebClient request headers.

WebClient client = new WebClient();
client.Headers.put("DecibelAppID", "<Your Application ID>");
client.Headers.put("DecibelAppKey", "<Your Application Key>");
client.Headers.put("DecibelTimestamp", new SimpleDateFormat("yyyyMMdd HH:mm:ss").format(new Date()));

The downloadStringAsync(Uri) method downloads a String from a resource, without blocking the calling thread. This enables the application to continue doing useful work while the method call runs.

The DownloadStringCompleted event is raised each time an asynchronous operation to download a resource as a string completes. The DownloadStringCompletedEvent class provides the event handler with event data.

The following code example request albums by artist name and stores the XML response.

WebClient client = new WebClient();
Util.setDecibelRequestHeaders(client, "<Your Application ID>", "<Your Application Key>");
client.addWebClientEventListener(new WebClientListener() {
    @Override
    public void downloadStringCompleted(DownloadStringCompletedEvent evt) {
        String result;
        if (evt.Error == null) {
            result = evt.Result;
        }
    }
});
client.downloadStringAsync("http://api.decibel.net/v1/Albums/?artist="
        + "=miles%20davis&format=xml");

Object Model
Back to top

The Decibel REST API returns data in two formats XML and JSON. To make it easier to parse the response the data can be deserialized into manageable objects.

To deserialize the Decibel REST API responses you will need to add the queryresultholder.jar package to your project. This file contains the class definitions for the different types of objects which can be requested from the API.

  • Right click on the project name in the project explorer
  • Select add new folder and name this libs
  • Download the queryresultholder.jar class (Right click and select Save link as...)
  • Copy the queryresultholder.jar file to the libs folder
  • In Eclipse right click the project name and select Build Path->Add External Archives...
  • In NetBeans right click on the Libraries folder and select Add JAR/Folder...
  • Select the queryresultholder.jar file and click Add

XML Deserialization
Back to top

JAXB short for Java Architecture for XML Binding helps transform XML into Java objects and vice versa using Java Annotations. This is done with the help of the Marshaller class which is responsible for the process of deserializing XML data into Java objects.

The following code example deserializes the XML response from a Albums API call to a AlbumQueryResult object using JAXB.

public void downloadStringCompleted(DownloadStringCompletedEvent evt) {

    if (evt.Error == null) {

        // Convert result string to InputSource
        InputSource is = new InputSource(new StringReader(evt.Result));

        AlbumQueryResult queryResult;
        try
        {
            // Create a JAXB context and unmarshaller
            JAXBContext context = JAXBContext.newInstance(AlbumQueryResult.class);
            Unmarshaller handler = context.createUnmarshaller();

            // Deserialize the XML stream 
            queryResult = (AlbumQueryResult)handler.unmarshal(is);
        }
        catch(JAXBException e) { }
    }
}    

To make it easier to deserialize the different types of XML response objects a generic function is defined in the Util.java class included with the sample project.

The following example code deserializes a Tracks API call to a TrackQueryResult object.

TrackQueryResult queryResult = Util.<TrackQueryResult>deserializeXmlString(TrackQueryResult.class, evt.Result);

private void beginAlbumSearch(String albumTitle, String applicationID, String applicationKey)
        throws UnsupportedEncodingException, NullPointerException  {

    // Set the result table model
    tblAlbumResults.setModel(new AlbumTableModel());

    // Construct the WebClient and set the request headers
    WebClient client = new WebClient();
    client.Headers.put("DecibelAppID", applicationID);
    client.Headers.put("DecibelAppKey", applicationKey);
    client.Headers.put("DecibelTimestamp",
            new SimpleDateFormat("yyyyMMdd HH:mm:ss").format(new Date()));

    // Set the callback function which is invoked when the download of the resource completes
    client.addWebClientEventListener(new WebClientListener() {
        @Override
        public void downloadStringCompleted(DownloadStringCompletedEvent evt) {

            if (evt.Error == null) {

                // Deserialize the response 
                AlbumQueryResult queryResult;
                try {
                    queryResult = Util.<AlbumQueryResult>deserializeXmlString(
                            AlbumQueryResult.class, evt.Result);
                } catch (JAXBException e) {
                    JOptionPane.showMessageDialog(null, "Unable to deserialize object "
                            + "from response body." + System.lineSeparator() + e.toString());
                    return;
                }

                // Initialize the results table
                DefaultTableModel model = (DefaultTableModel) tblAlbumResults.getModel();
                List<Album> albums = queryResult.getResultSet().getAlbum();
                for (Album album : albums) {
                    Object[] row = new Object[5];
                    row[0] = album.getID();
                    row[1] = album.getName();
                    row[2] = album.getArtists();
                    row[3] = Util.formatTime(album.getTotalSeconds().intValue());
                    row[4] = album.getTrackCount();
                    model.addRow(row);
                }
                tblAlbumResults.setModel(model);

            } else {
                JOptionPane.showMessageDialog(null, "The Decibel Web Service request "
                        + "was not successful due to the following error:"
                        + System.lineSeparator() + evt.Error.toString());
            }
        }
    });

    // Issue the request to the Decibel Web Service
    client.downloadStringAsync("http://api.decibel.net/v1/Albums/?albumTitle="
            + URLEncoder.encode(albumTitle, "UTF-8") + "&format=xml");
}

private class AlbumTableModel extends DefaultTableModel {
    public AlbumTableModel() {
        addColumn("ID");
        addColumn("Name");
        addColumn("Artists");
        addColumn("Length");
        addColumn("Track Count");
    }
}

private void beginAlbumRetrieve(String albumID, String applicationID, String applicationKey) {
    
    // Construct the WebClient and set the request headers
    WebClient client = new WebClient();
    client.Headers.put("DecibelAppID", applicationID);
    client.Headers.put("DecibelAppKey", applicationKey);
    client.Headers.put("DecibelTimestamp",
            new SimpleDateFormat("yyyyMMdd HH:mm:ss").format(new Date()));

    // Set the callback function which is invoked when the download of the resource completes
    client.addWebClientEventListener(new WebClientListener() {
        @Override
        public void downloadStringCompleted(DownloadStringCompletedEvent evt) {
            
            if(evt.Error == null)
            {
                // Deserialize the response 
                AlbumQueryResult queryResult;
                try {
                    queryResult = Util.<AlbumQueryResult>deserializeXmlString(
                            AlbumQueryResult.class, evt.Result);
                } catch (JAXBException e) {
                    JOptionPane.showMessageDialog(null, "Unable to deserialize object "
                            + "from response body." + System.lineSeparator() + e.toString());
                    return;
                }
                                
                // Display the album information
                Album album = queryResult.getResultSet().getAlbum().get(0);
                txtName.setText(album.getName());
                txtArtists.setText(album.getArtists());
                txtDiscs.setText(Integer.toString(album.getDiscCount()));
                txtTracks.setText(Integer.toString(album.getTrackCount()));
                txtDuration.setText(Util.formatTime(album.getTotalSeconds().intValue()));
                if (album.getDiscs() != null && album.getDiscs().getDisc().size() > 0 && 
                        album.getDiscs().getDisc().get(0).getMusicMedium() != null)
                    txtFormat.setText(album.getDiscs().getDisc().get(0).getMusicMedium().getName());
                if (album.getReleaseDate() != null)
                    txtReleaseDate.setText(album.getReleaseDate().getName());
                if (album.getPublications() != null && album.getPublications().getPublication().size() > 0)
                    txtCatalogue.setText(album.getPublications().getPublication().get(0).getCatalogueNumber());
                if (album.getPublications() != null && album.getPublications().getPublication().size() > 0)
                    txtPublisher.setText(album.getPublications().getPublication().get(0).getPublisher().getName());
                if (album.getGenres() != null)
                {
                    for (GenreValue genreValue : album.getGenres().getGenreValue())
                    {
                        if (genreValue.getGenre() != null)
                            txtGenres.setText(txtGenres.getText() + genreValue.getGenre().getName() + ", ");
                    }
                }
                if (album.getGeoEntities() != null)
                {
                    for (GeoEntityValue genreEntityValue : album.getGeoEntities().getGeoEntityValue())
                    {
                        if (genreEntityValue.getGeoEntity() != null)
                            txtRegion.setText(txtRegion.getText() + genreEntityValue.getGeoEntity().getName() + ", ");
                    }
                }
                if (album.getExternalIdentifiers() != null)
                {
                    for (ExternalIdentifier externalIdentifier : album.getExternalIdentifiers().getExternalIdentifier())
                    {
                        ExternalDatabase externalDatabase = externalIdentifier.getExternalDatabase();
                        if (externalDatabase.getName().equals("UPC Barcode"))
                        {
                            txtBarcode.setText(externalIdentifier.getIdentifier());
                            break;
                        }
                    }
                }

                // Thumbnail image
                InputStream in = new ByteArrayInputStream(album.getThumbnail());
                try {
                    mImage = ImageIO.read(in);
                } catch (IOException e) {
                    e.printStackTrace();
                }
                repaint();   
                
                // Add the tracks to a list
                if (album.getTracks() != null)
                {
                    DefaultListModel list = new DefaultListModel();
                    for (Track track : album.getTracks().getTrack()) {
                            String trackInfo = track.getSequenceNo() + ") " + track.getName();
                            int duration = track.getTotalSeconds().intValue();
                            if (duration > 0) {
                                trackInfo += " (" + Util.formatTime(duration) + ")";
                            }
                            list.addElement(trackInfo);
                    }
                    lstTracks.setModel(list);
                }                  
            }
            else {
                JOptionPane.showMessageDialog(null, "The Decibel Web Service request "
                        + "was not successful due to the following error:"
                        + System.lineSeparator() + evt.Error.toString());
            }
        }
    });

    // Issue the request to the Decibel Web Service
    String depth = "Tracks;ImageThumbnail;Media;Names;ExternalIdentifiers;Genres;Publications;Performances;";
    client.downloadStringAsync("http://api.decibel.net/v1/Albums/" + "?depth=" + depth + "&id="
            + albumID + "&format=xml");
}

@Override
public void paint(Graphics g) {
    super.paint(g);
    // Draw the thumbnail image

    g.drawImage(mImage, mRect.x, mRect.y, mRect.width, mRect.height, this);
}

private void beginAlbumArtRetrieve(final String albumID, final String applicationID, final String applicationKey) {

    new Thread(new Runnable() {
        @Override
        public void run() {

            String error = null;
            try {
                URLConnection urlConnection = new URL("http://api.decibel.net/v1/Albums/"
                        + albumID + "/Image").openConnection();

                urlConnection.setRequestProperty("DecibelAppID", applicationID);
                urlConnection.setRequestProperty("DecibelAppKey", applicationKey);
                urlConnection.setRequestProperty("DecibelTimestamp",
                        new SimpleDateFormat("yyyyMMdd HH:mm:ss").format(new Date()));

                mImage = ImageIO.read(urlConnection.getInputStream());
                repaint();
            } catch (MalformedURLException ex) {
                error = ex.toString();
            } catch (IOException e) {
                error = e.toString();
            }

            if (error != null) {
                JOptionPane.showMessageDialog(null, "The Decibel Web Service request "
                        + "was not successful due to the following error:"
                        + System.lineSeparator() + error);
            }
        }
    }).start();
}

@Override
public void paint(Graphics g) {
    g.drawImage(mImage, 0, 25, this.getWidth(), this.getHeight(), this);
}

public void beginAutoCompleteRequest(String text, final String dictionary) 
        throws UnsupportedEncodingException
{ 
    // Construct the WebClient and set the request headers
    WebClient client = new WebClient();
    client.Headers.put("DecibelAppID", "<Your Application ID>");
    client.Headers.put("DecibelAppKey", "<Your Application Key>");
    client.Headers.put("DecibelTimestamp",
            new SimpleDateFormat("yyyyMMdd HH:mm:ss").format(new Date()));

    // Set the callback function which is invoked when the download of the resource completes
    client.addWebClientEventListener(new WebClientListener() {
        @Override
        public void downloadStringCompleted(DownloadStringCompletedEvent evt) {

            if (evt.Result != null) {

                // Deserialize the response 
                SuggestionQueryResult suggestionQueryResult;
                try {
                    suggestionQueryResult = Util.<SuggestionQueryResult>deserializeXmlString(
                            SuggestionQueryResult.class, evt.Result);
                } catch (JAXBException e) {
                    return;
                }

                // Add the suggestion results to the data list
                ArrayList<String> data = new ArrayList<>();
                switch (dictionary) {
                    case "Albums":
                        for (Suggestion suggestion : suggestionQueryResult.getAlbums().getSuggestion()) {
                            data.add(suggestion.getSuggestionValue());
                        }
                        break;
                    case "Participants":
                        for (Suggestion suggestion : suggestionQueryResult.getParticipants().getSuggestion()) {
                            data.add(suggestion.getSuggestionValue());
                        }
                        break;
                    case "Tracks":
                        for (Suggestion suggestion : suggestionQueryResult.getTracks().getSuggestion()) {
                            data.add(suggestion.getSuggestionValue());
                        }
                        break;
                }
            }
        }
    });
    client.downloadStringAsync("http://api.decibel.net/v1/AutoComplete?text="
            + URLEncoder.encode(text, "UTF-8") + "&dictionary=" + dictionary + "&limit=5");        
}
